# HTML Forms

## Introduction

HTML forms are used to collect input from users on a web page. The information entered by the user can be sent to a server for processing. Forms are commonly used for registration, login, surveys, and feedback.

## Form Tag

The `<form>` tag is used to create a form in HTML.

Example:

```html
<form>
  <!-- form elements go here -->
</form>
```

## Input Elements

The `<input>` tag is used to create different types of input fields.

Common input types include:

* text
* password
* email
* number
* radio
* checkbox
* submit
* reset

Example:

```html
<input type="text" placeholder="Enter your name">
```

## Label

The `<label>` tag is used to describe an input field.

Example:

```html
<label>Name:</label>
<input type="text">
```

## Textarea

The `<textarea>` tag allows users to enter multiple lines of text.

Example:

```html
<textarea rows="4" cols="30"></textarea>
```

## Select (Dropdown)

The `<select>` tag creates a dropdown list.

Example:

```html
<select>
  <option>India</option>
  <option>USA</option>
</select>
```

## Buttons

Buttons are used to submit or reset the form.

Example:

```html
<input type="submit" value="Submit">
<input type="reset" value="Reset">
```

## Conclusion

HTML forms are important for collecting user information on websites. By using different form elements like input fields, labels, dropdowns, and buttons, developers can create interactive and user-friendly web pages.
